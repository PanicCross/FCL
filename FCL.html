How to edit it

You only need to change the numbers and names inside the table rows.

Update the drivers table

Open the HTML file in any code editor and find the first <tbody> section under Drivers Championship. Each row is one driver, for example:

<tr><td>1</td><td class="driver">David Burke10</td>...</tr>

Add team standings later

When you are ready, replace the placeholder rows in the Team Championship table with your real teams and points. The layout is already there, so you do not need to redesign the page.

  • Change positions and points manually.
  • Add or remove rows as needed.
  • Duplicate an existing <tr> row to add another entry.
